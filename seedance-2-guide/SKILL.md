# Seedance 2.0 · 参考手册

> 本技能是**知识型 Skill**，不调接口，不写代码。当你需要判断 Seedance 2.0 的能力边界、选择工作模式、理解提示词结构时查阅本文档。真正的生成调用走 `libtv-skill`。

## ⚠️ 与 LibTV Skill 的协作原则（重要）

`libtv-skill` 的核心原则是：**用户侧不改写 prompt，只做传话**。后端 Agent 会自行处理 prompt 工程、模型选择、参数配置。

**本手册的正确用法：**
- ✅ 帮你/用户**判断**当前需求是否适合 Seedance 2.0（vs 需要改用 Kling O3）
- ✅ 帮你**回答**用户关于模型能力、参数范围、@符号、工作模式的问题
- ✅ 在用户主动索要模板时，**展示**提示词模板供用户挑选和二次创作
- ✅ 帮你**诊断**生成失败时的可能原因（如人物换脸 → 提示用户提供更清晰的参考图）

**绝对不要：**
- ❌ 拿本手册里的提示词模板**偷偷嫁接**到用户请求中再发给 libtv-skill（违反传话原则）
- ❌ 替用户扩写为「@character1, cinematic, 8K, 24fps...」之类的技术参数
- ❌ 擅自判断用户应该用哪个模板并直接执行

本手册是给**人类创作者**和**模型选型决策**用的，不是给 Agent 用来注入 prompt 的。

## 一、Seedance 2.0 定位

基于扩散模型的 AI 视频生成工具，主打「**参考视频 + 文本提示词**」的可控生成。

**核心优势：**
- **角色一致性**：解决 AI 视频"换脸""换衣"的痛点，同一角色可贯穿多镜头
- **镜头语言可控**：精准控制运镜、转场、节奏，媲美专业剪辑
- **成本门槛低**：相比传统影视制作，时间和资金成本大幅降低
- **商业场景成熟**：电商广告、短剧、MV、品牌营销等领域已有大量成功案例

**适合的场景：**
- 电商 TVC、产品广告、探店 / 测评短视频
- 短剧、漫剧、微电影、品牌故事片
- 音乐 MV（支持音乐卡点）
- 旅游探险、生活方式类内容
- 多镜头叙事、同角色贯穿全片

**不擅长的场景：**
- 超长篇叙事（剧情复杂度 > 3 分钟的故事）
- 极端抽象 / 超现实实验艺术（Sora 更合适）
- 纯速度优先的 UGC 短视频（Veo 更便宜更快）

## 二、两种工作模式

### 模式 1：首尾帧模式（新手必选）

- **适用场景**：快速生成简单转场、测试创意、验证提示词效果
- **操作流程**：
  1. 上传首帧（起始画面）+ 尾帧（目标画面）
  2. 输入提示词描述中间过渡过程
  3. 调整时长（推荐 **2-5 秒**）和运动强度
- **优势**：上手快，适合验证创意，无需复杂参考视频

### 模式 2：全能参考模式（进阶必选）

- **适用场景**：多镜头叙事、角色一致性控制、复杂运镜
- **操作流程**：
  1. 上传 **1-3 段参考视频**（建议 10-30 秒，清晰稳定）
  2. 为每段视频绑定提示词，定义镜头内容和风格
  3. 开启「角色一致性」开关，锁定核心人物
- **优势**：高度可控，支持多场景无缝衔接，是**商业创作的核心模式**

## 三、@ 符号 · 核心控制符

@ 是 Seedance 2.0 的**核心控制符**，用于精确绑定参考视频中的元素（角色、物体、场景），让模型理解「这是什么」「要做什么」，避免生成混乱。

### 常见用法

| 用法 | 示例 | 作用 |
|------|------|------|
| **锁定角色** | `@character1` | 绑定参考视频中的主角，确保多镜头一致性 |
| **指定物体** | `@product` | 绑定商品，保证电商广告中产品细节不变 |
| **定义场景** | `@cityscape` | 绑定城市背景，维持场景连贯性 |
| **控制动作** | `@character1 walks forward` | 让角色执行特定动作 |

### 实战案例

**电商广告**：
```
@product is a sleek wireless earbud, rotating slowly on a marble countertop,
soft studio lighting, 8K, product photography
```

**短剧转场**：
```
@character1 opens a door, revealing a neon-lit cyberpunk alley,
rain falling, cinematic dolly shot
```

## 四、提示词「黄金公式」

```
[核心主体] + [动作/场景] + [风格/镜头] + [技术参数]
```

### 拆解说明

| 部分 | 作用 | 示例 |
|------|------|------|
| **核心主体** | 明确角色、物体或场景 | `@character1, a vintage motorcycle` |
| **动作 / 场景** | 描述主体行为和环境 | `driving through a desert at sunset, standing in a rain-soaked street` |
| **风格 / 镜头** | 定义视觉风格和运镜 | `cinematic, 35mm lens, shallow depth of field` |
| **技术参数** | 控制画质、帧率、时长 | `8K, 24fps, 4 seconds, smooth motion` |

### 完整示例

**电商类**：
```
@wireless_headphones, floating in mid-air, soft golden hour light,
transparent background, product photography, 8K, 30fps, 3 seconds
```

**短剧类**：
```
@character1, a young detective, examining a bloodstained note in a dimly lit office,
low-key lighting, Dutch angle, suspenseful, 2K, 24fps, 5 seconds
```

## 五、六大进阶技巧

| # | 技巧 | 原理 | 应用场景 |
|---|------|------|---------|
| 1 | **九宫格分镜法** | 将视频拆解为 9 个关键帧，逐一设计提示词，确保叙事流畅 | 短剧、MV、品牌宣传片 |
| 2 | **分场景控制法** | 为每个场景单独设置参考视频和提示词，精准控制视觉风格和节奏 | 电商广告、旅游视频、多场景短剧 |
| 3 | **多模态融合法** | 结合文本、图像、音频多模态输入，让视频更具沉浸感 | 音乐 MV、沉浸式广告、互动视频 |
| 4 | **导演分镜节点法** | 模仿电影分镜，在关键节点（转场、高潮）精准控制镜头和情绪 | 影视剧、品牌大片、剧情类广告 |
| 5 | **参考视频克隆法** | 克隆专业运镜视频的镜头语言，快速提升视频质感 | 电影级广告、MV、剧情短片 |
| 6 | **情感递进法** | 通过光线、色彩、运镜逐步强化情感，引导观众情绪 | 情感类广告、微电影、品牌故事片 |

## 七、模型对比速查

| 工具 | 核心优势 | 适用场景 | 成本 |
|------|---------|---------|------|
| **Seedance 2.0** | 角色一致性强、可控性高、参考视频支持 | 商业广告、短剧、MV | 中等 |
| **Sora** | 生成质量高、叙事能力强 | 电影短片、创意实验 | 高 |
| **Veo** | 速度快、操作简单 | 社交媒体短视频 | 低 |

**结论**：Seedance 2.0 是目前**商业创作性价比最高**的选择，尤其适合需要角色一致性和多场景控制的项目。

## 九、AI 导演成长路径

从简单开始：先用**首尾帧模式**测试创意，再逐步进阶到**全能参考模式**。

1. **积累参考库**：收集优质运镜、场景、角色视频，作为后续创作的参考素材
2. **迭代提示词**：不断优化提示词，结合 **@ 符号** 和 **黄金公式**，精准控制生成结果
3. **拥抱商业应用**：从电商广告、短剧等低门槛场景入手，逐步拓展到更复杂的影视项目
