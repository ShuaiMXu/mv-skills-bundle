# 主题色预设（Themes）

**5 套精心调配的主题色板**，保证"电子杂志 × 电子墨水"的美学不垮。
**不允许用户自定义颜色** —— 色彩搭配错了画面瞬间变丑，只从以下预设中挑选。

> 源自 op7418 guizang-ppt-skill，已验证可用于公众号、PPT、小红书等多种输出形态。

---

## 使用方法

1. 问用户选哪套（或基于内容推荐一套）
2. 打开下游 skill 的 `assets/template.html` 的 `<style>` 块
3. 找到开头的 `:root{` 块
4. **整体替换**以下 CSS 变量：
   - `--ink` / `--ink-rgb`：主文字色（深色）
   - `--paper` / `--paper-rgb`：背景色（浅色）
   - `--paper-tint`：次级背景（深一度）
   - `--ink-tint`：次级文字（浅一度）
5. 所有其他 CSS 都走 `var(--...)`，无需其他改动

## 🖋 墨水经典 (Monocle 默认)

**适合**：通用分享、商业发布、科技产品、任何场景都安全的默认选择。
**调性**：纯墨黑 + 暖米白，杂志感最强，Monocle / Apricot / A Book Apart 风。

```css
--ink:#0a0a0b;
--ink-rgb:10,10,11;
--paper:#f1efea;
--paper-rgb:241,239,234;
--paper-tint:#e8e5de;
--ink-tint:#18181a;
```

---

## 🌊 靛蓝瓷 (Indigo Porcelain)

**适合**：科技/研究/数据分享、工程师文化、深度内容、技术发布会。**AI 日报首选**。
**调性**：深靛蓝 + 瓷白，冷静、理性、有深度，像学术期刊或蓝印花瓷器。

```css
--ink:#0a1f3d;
--ink-rgb:10,31,61;
--paper:#f1f3f5;
--paper-rgb:241,243,245;
--paper-tint:#e4e8ec;
--ink-tint:#152a4a;
```

## 🌿 森林墨 (Forest Ink)

**适合**：自然/可持续/文化/非虚构内容、户外品牌、环保主题。
**调性**：深森林绿 + 象牙，沉稳、有呼吸感，像旧版《国家地理》。

```css
--ink:#1a2e1f;
--ink-rgb:26,46,31;
--paper:#f5f1e8;
--paper-rgb:245,241,232;
--paper-tint:#ece7da;
--ink-tint:#253d2c;
```

---

## 🍂 牛皮纸 (Kraft Paper)

**适合**：怀旧/人文/阅读/历史/文学分享、独立杂志、手作品牌。
**调性**：深棕 + 暖米，像牛皮信封或老笔记本，温暖、有年代感。

```css
--ink:#2a1e13;
--ink-rgb:42,30,19;
--paper:#eedfc7;
--paper-rgb:238,223,199;
--paper-tint:#e0d0b6;
--ink-tint:#3a2a1d;
```

## 🌙 沙丘 (Dune)

**适合**：艺术/设计/创意/时尚分享、画廊手册、审美优先的私享会。
**调性**：炭灰 + 沙色，克制、高级、中性，像沙漠黄昏或建筑设计图册。

```css

--paper:#f0e6d2;
--paper-rgb:240,230,210;
--paper-tint:#e3d7bf;
```

## 切换原则

- **一份内容只用一套主题**，不要中途换色
- `currentColor` 驱动的 border / icon 会跟随文字颜色自动适配，无需额外调整
- 选定主题后，标题文字和装饰文案可以强化该主题的语义（例如牛皮纸配"Vol.03 · 秋"这种）

---

## 公众号媒介特别说明

**公众号 HTML 不支持 JavaScript 和 WebGL**，原 guizang skill 中的 shader 背景不适用。公众号场景下：
- 只用 CSS 变量 + 纯 CSS 渐变替代 WebGL 背景
- `currentColor`、`var(--...)` 在公众号 HTML 完全支持
- 避免 `position:fixed`（公众号编辑器会重排）
- 图标建议用 **CSS 伪元素 + emoji** 或 **内联 SVG**（不要外链 SVG）

---

## ❌ 不要做的事

- ❌ **不允许混搭**（例如 ink 取墨水经典的，paper 取沙丘的）——会彻底违和
- ❌ **不允许用户随便给一个 hex 值**——需委婉拒绝并展示 5 套预设让选
- ❌ **不要直接修改 template.html 其他地方的颜色**——所有散落 rgba 都走 var，改 :root 一处即可

选定主题后在对话中告诉用户："用 🖋 墨水经典 / 🌊 靛蓝瓷 ..."并在项目记录里备注，方便后续迭代时保持一致。
